## 简单配置

用vim编辑 ~/.vimrc , 我这里没有，直接创建一个

	set tabstop=4
	set softtabstop=4
	set shiftwidth=4
	set expandtab

	set nu		#设置行号
	set autoindent	#设置自动缩进
	syntax on 	#设置语法高亮
